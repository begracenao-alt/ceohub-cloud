# Googleスプレッドシート連携の手順

アプリのデータを、自動でGoogleスプレッドシートにバックアップ・閲覧できるようにする設定です。
**やらなくてもアプリは普通に使えます**（データはブラウザ内に保存されます）。
「スプレッドシートでも見たい・残したい」ときだけ、この手順を一度だけ行ってください。

所要時間：約5〜10分。むずかしくありません。順番に進めれば大丈夫です🤍

---

## 1. スプレッドシートを用意する

1. [https://sheets.google.com](https://sheets.google.com) を開く
2. 「空白のスプレッドシート」を新規作成
3. 名前を「Be Grace CEO Hub データ」などにする

## 2. スクリプトを貼り付ける

1. メニューの **「拡張機能」→「Apps Script」** をクリック
2. 開いたエディタの中身（`function myFunction() {}` など）を**全部消す**
3. このフォルダの **`Code.gs`** の中身を**全部コピーして貼り付ける**
4. 上の 💾（保存）アイコンをクリック

## 3. ウェブアプリとして公開する

1. 右上の **「デプロイ」→「新しいデプロイ」** をクリック
2. 「種類の選択」の⚙️マーク →​ **「ウェブアプリ」** を選ぶ
3. 設定：
   - 説明：`Be Grace CEO Hub`（何でもOK）
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**全員**
4. **「デプロイ」** をクリック
5. 「アクセスを承認」を求められたら → 自分のGoogleアカウントを選ぶ
   - 「このアプリはGoogleで確認されていません」と出たら →
     **「詳細」→「（プロジェクト名）に移動」→「許可」** で進めます（自分のスクリプトなので安全です）
6. 表示された **「ウェブアプリのURL」**（`https://script.google.com/macros/s/.../exec`）を**コピー**

## 4. アプリ側に貼る

1. Be Grace CEO Hub を開く
2. 左メニュー **「⚙️ 初期設定・保存」** を開く
3. 「📊 Googleスプレッドシート連携」の入力欄に、さっきコピーした**URLを貼り付け**
4. **「連携をオンにする」** をクリック

これで完了です。
以降、アプリでデータを保存するたびに、自動でスプレッドシートのタブ（売上・経費・顧客…）に書き込まれます。

---

## よくある質問

**Q. スプレッドシートを編集したらアプリに反映される？**
A. いいえ。連携は「アプリ → スプレッドシート」の一方向です。スプレッドシートは閲覧・バックアップ用とお考えください。

**Q. うまく動かない**
A. URLが `…/exec` で終わっているか確認してください。それでもダメなら、いったん連携をオフにして、アプリの「📥 バックアップを保存」でファイルに残せば安心です。

**Q. データが消えないか心配**
A. いちばん確実なのは、設定画面の **「📥 バックアップを保存（ファイル）」** です。ときどき押してファイルを保存しておけば、何があっても復元できます。
